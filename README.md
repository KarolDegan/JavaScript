## 🎓 Conteúdo Detalhado

### 📍 Alura - Criando Requisições

Projeto desenvolvido durante o curso **"JavaScript: criando requisições"** da plataforma Alura.

**O que foi aprendido:**
- Fetch API e requisições HTTP
- JSON Server para simular backend
- Manipulação de respostas assíncronas
- Tratamento de erros em requisições
- Integração frontend com API

**Tecnologias:**
- JavaScript moderno (Fetch API)
- NodeJS
- json-server
- HTML5 e CSS3

**Estrutura do projeto:**

```javascript
// Requisição GET com Fetch API
fetch('http://localhost:3000/videos')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Erro:', error));

// Requisição com async/await
async function obterVideos() {
  try {
    const response = await fetch('http://localhost:3000/videos');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Erro:', error);
  }
}
```

Estudos aprofundados sobre **TypeScript**, linguagem que adiciona tipagem estática ao JavaScript.

**Conceitos cobertos:**

#### Tipagem Básica
```typescript
// Tipos primitivos
let nome: string = "Karol";
let idade: number = 25;

let numeros: number[] = [1, 2, 3];
```

#### Interfaces
```typescript
interface Usuario {
  id: number;
  nome: string;
  email: string;
  readonly dataCadastro: Date;
}

const usuario: Usuario = {
  id: 1,
  nome: "Karol",

  dataCadastro: new Date()

```

#### Genéricos
```typescript
// Função genérica
function processar<T>(valor: T): T {
  return valor;
}

// Tipo genérico com restrição
function obterPropriedade<T, K extends keyof T>(obj: T, chave: K): T[K] {
  return obj[chave];
}
```

```typescript

  console.log(`Classe ${constructor.name} foi criada`);
```

### 3️⃣ Programação Assíncrona

**Callbacks (padrão antigo):**

```javascript
function buscarDados(callback) {
  setTimeout(() => {
    callback(null, { id: 1, nome: 'Karol' });
  }, 1000);
}

buscarDados((erro, dados) => {
  if (erro) console.error(erro);
  console.log(dados);
});
```

**Promises (padrão intermediário):**

```javascript
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve({ id: 1, nome: 'Karol' });
  }, 1000);
});

promise
  .then(dados => console.log(dados))
  .catch(erro => console.error(erro))
  .finally(() => console.log('Finalizado'));
```

**Async/Await (padrão moderno):**

```javascript
async function buscarDados() {
  try {
    const response = await fetch('/api/usuarios');
    const dados = await response.json();
    return dados;
  } catch (erro) {
    console.error('Erro:', erro);
  }
}

// Usando
const dados = await buscarDados();
```

### 4️⃣ Orientação a Objetos

**Classes:**

```typescript
class Pessoa {
  private nome: string;
  private idade: number;

  constructor(nome: string, idade: number) {
    this.nome = nome;
    this.idade = idade;
  }

  get apresentacao(): string {
    return `Olá, meu nome é ${this.nome}`;
  }

  apresentar(): void {
    console.log(this.apresentacao);
  }
}

const pessoa = new Pessoa('Karol', 25);
pessoa.apresentar(); // Olá, meu nome é Karol
```

**Herança:**

```typescript
class Funcionario extends Pessoa {
  private salario: number;

  constructor(nome: string, idade: number, salario: number) {
    super(nome, idade);
    this.salario = salario;
  }

  ganharAumento(percentual: number): void {
    this.salario *= (1 + percentual / 100);
  }
}
```

### 5️⃣ Consumo de APIs

**Com Fetch API:**

```javascript
// GET
async function obterUsuarios() {
  const response = await fetch('https://api.exemplo.com/usuarios');
  const usuarios = await response.json();
  return usuarios;
}

// POST
async function criarUsuario(dados) {
  const response = await fetch('https://api.exemplo.com/usuarios', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(dados)
  });
  return await response.json();
}

// PUT
async function atualizarUsuario(id, dados) {
  const response = await fetch(`https://api.exemplo.com/usuarios/${id}`, {
    method: 'PUT',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(dados)
  });
  return await response.json();
}

// DELETE
async function deletarUsuario(id) {
  const response = await fetch(`https://api.exemplo.com/usuarios/${id}`, {
    method: 'DELETE'
  });
  return response.ok;
}
```
