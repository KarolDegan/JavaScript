## 🎯 Visão Geral

Este repositório tem como objetivo documentar meu aprendizado em JavaScript e tecnologias relacionadas, com foco na prática e consolidação de conceitos fundamentais e intermediários do desenvolvimento web.

O conteúdo inclui:

* Exercícios de lógica de programação
* Manipulação do DOM
* Consumo de APIs
* Programação assíncrona
* Estudos de TypeScript
* Aplicação de boas práticas de organização de código
* Exercícios baseados em cursos da Alura

## 🛠️ Tecnologias Utilizadas

### Linguagens de Programação

| Tecnologia            | Aplicação                                   |
| --------------------- | ------------------------------------------- |
| **JavaScript (ES6+)** | Lógica, interatividade e manipulação do DOM |
| **TypeScript**        | Tipagem estática e organização de código    |
| **HTML5**             | Estrutura semântica de páginas              |
| **CSS3**              | Estilização e responsividade                |

### Recursos e APIs

| Tecnologia      | Aplicação                             |
| --------------- | ------------------------------------- |

| **Async/Await** | Controle de operações assíncronas     |
| **Promises**    | Tratamento de chamadas assíncronas    |
| **JSON**        | Estrutura e troca de dados            |

| Ferramenta      | Uso                                      |
